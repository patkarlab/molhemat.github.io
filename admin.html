Site Manager 🔬

Manage the team, publications, tools, links and lab photos of molhemat.org — changes are saved straight to the website, no coding needed.

How do I get a token? (one-time setup, 2 minutes)
  1. Sign in to GitHub as an account with write access to patkarlab/molhemat.github.io.
  2. Go to github.com/settings/personal-access-tokens/new.
  3. Name it e.g. molhemat site manager, set a long expiry.
  4. Under Repository access choose Only select repositories → patkarlab/molhemat.github.io.
  5. Under Permissions → Repository permissions set Contents to Read and write.
  6. Click Generate token, copy it, and paste it above.

The token never leaves your browser — it talks directly to GitHub.
